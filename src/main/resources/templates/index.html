General Information

This website was developed to demonstrate my Java Developer skills. It combines both Spring and Java EE technologies: navigation between pages and forms are processed using Spring mappings, and all interactions with the underlying database tables are done using Spring Data JPA and JAX-RS. Information about inventory items and users with relevant priviliges is stored in the relevant tables. Some pages are protected using Spring Security/Java EE technologies.

The following technologies were used to develop the website:

  1. Java 8
  2. Spring Core
  3. Spring MVC
  4. Spring Data JPA
  5. JAX-RS
  6. Thymeleaf (pages are built using specific fragments like Apache Tiles)
  7. Tomcat
  8. MySQL
